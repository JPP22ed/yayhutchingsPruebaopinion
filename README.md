# Pruebaopinion - Poemas de Yay Hutchings

Este proyecto muestra poemas de Yay Hutchings con un diseño interactivo. Cada poema se puede expandir y contraer con un click.

## Autor
Yay Hutchings

## Libro Referenciado (APA 7)
Hutchings, Y. (2024). *Viviendo de dolor*. Circulo Rojo.

## Despliegue
Se puede desplegar directamente en Vercel como sitio estático.
